
# Rhetenor VS Code Extension

Integration of Rhetenor backtesting engine and Butterflow strategy language for VS Code.

## Features

-   **Butterflow Language Support**: Syntax highlighting for `.bf` files.
-   **Backtest Execution**: Run backtests directly from VS Code.
-   **Dynamic Visualization**: View interactive PnL and signal charts.

## Usage

1.  Open a `.bf` file.
2.  Run command `Rhetenor: Run Backtest`.

## Requirements

-   Python 3.10+
-   `rhetenor` python package installed.
